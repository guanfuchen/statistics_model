## 数据操作
在深度学习中，我们通常会频繁地对数据进行操作。作为动手学深度学习的基础，本节将介绍如何对内存中的数据进行操作。
在MXNet 中，NDArray 是存储和变换数据的主要⼯具。如果你之前用过NumPy，你会发现NDArray和NumPy 的多维数组非常类似。然而，NDArray 提供诸如GPU 计算和自动求导在内的更多功能。这些都使得NDArray 更加适合深度学习。
